Public Class methods

new(cfg={})

Set the parameters of this class. The cfg hash must contain:

  • :l => array with the 4 length of arm
  • :home => the base position of joints
  • :limits => the work‘s range of every joints
  • :a => the a coefficients of Denavit-Hartenberg
  • :alpha => the alpha coefficients of Denavit-Hartenberg
  • :d => the d coefficients of Denavit-Hartenberg

Public Instance methods

ik(pose=@pose)

Update the joints values for reach the pose position of end effector. The pose hash must contains:

  • :x => x position of end effector
  • :y => y position of end effector
  • :z => z position of end effector
  • :phi => angular position of end effector
